Interface Pen

A pen is the most basic tool you can think of for signing. It works everywhere and can be used intuitively by everyone. However, it does not come with a great amount of features. End of semi suitable metaphor.

This wraps a single keypair and allows for signing.

Non-goals of this types are: multi account support, persistency, data migrations, obfuscation of sensitive data.
